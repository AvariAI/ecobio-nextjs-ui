# Stat Boost Traits System - Simple Version

## Overview

**Simple design:** One trait slot for ALL ranks (E through S+).

**Personnalités (6):** 5 stat-focused + 1 random
**Traits (6):** 5 stat-focused + 1 random

---

## Personnalités

| Personnalité | Stat Focus | Multiplier |
|--------------|------------|------------|
| agressive | ATK | ×2.715 (+3.5% per level) |
| protective | DEF | ×2.715 |
| rapide | VIT | ×2.715 |
| soin_leurre | HP | ×2.715 |
| précise | CRIT | ×2.715 |
| mysterieuse | RANDOM | ×2.715 on RANDOM stat per level |

---

## Traits

| Trait | Stat Focus | Effet | Max @ 50 |
|-------|------------|--------|---------|
| Vitalité | HP | +0.6% HP/level | +29.4% |
| Puissance | ATK | +0.6% ATK/level | +29.4% |
| Carapace | DEF | +0.6% DEF/level | +29.4% |
| Vitesse | VIT | +0.6% VIT/level | +29.4% |
| Précision | CRIT | +0.6% CRIT/level | +29.4% |
| Mystère | RANDOM | +0.6% RANDOM/level (changes per level) | +29.4% |

---

## Slots

TOUS les rangs = **1 trait slot**

---

## Example Builds

| Personnalité + Trait | Result |
|---------------------|---------|
| agressive + Puissance | Max ATK (×3.51) |
| agressive + Carapace | High ATK + Medium DEF |
| protective + Puissance | High DEF + Medium ATK |
| mysterieuse + Mystère | Random + Random = FUN! |

**Personnalité:** Base archetype (what you naturally boost)
**Trait:** Customization (what you want additionally)

---

## Balance

All personalities have same 2.715× multiplier on their focus stat.
All traits have same 0.6% scaling per level.

**Equality:** Each has advantage on ONE stat.

No broken "all stats" personality → perfectly balanced.
